## 终端的基础命令
1. ls 列出当前路径下的所有文件
2. cd 进入文件夹
3. cd ..退回上一文件夹
4. rm -rf 删除操作
5. mkdir 创建文件夹
6. touch 创建文件
7. git clone 下载代码
8. git add . 提交代码
9. git commit -m "注释" 更新代码
11. git pull 
10. git push 推送代码
